Ulunaz.ai

Bursa Uludağ Üniversitesi İnegöl İşletme Fakültesi'nin yapay zeka kampüs asistanı.

Ulunaz (Nazlı), İnegöl İşletme Fakültesi kampüsünün sevilen köpeğinden esinlenen bir AI chatbot'tur. Öğrencilere fakülte, dersler, sınav takvimleri, topluluklar, barınma, ulaşım ve İnegöl şehri hakkında bilgi verir.

Özellikler
AI Chatbot - Gemini 2.0 Flash / DeepSeek destekli akıllı asistan
RAG Sistemi - Yerel bilgi tabanından anahtar kelime tabanlı arama
Sınav Takvimi - İşletme, UTI ve YBS bölümlerinin vize programı
Topluluk Bilgileri - UYBİST, İNİT, UATT topluluk detayları ve yönetim kadroları
Kampüs Rehberi - Barınma (KYK, İlim Yayma Cemiyeti), kantin, sosyal alanlar
İnegöl Şehir Rehberi - Gezilecek yerler, köfteciler, parklar, AVM'ler, belediye hizmetleri
Ulaşım Bilgileri - İnegöl otobüs hatları, Görükle kampüsüne metro rotası
Sohbet Geçmişi - localStorage tabanlı sohbet kaydı ve yönetimi
Fotoğraf Desteği - Ekip fotoğrafını chatbot içinde gösterme
Teknolojiler
Next.js (App Router)
TypeScript
Tailwind CSS
Google Gemini API / DeepSeek API
Vercel AI SDK
Framer Motion
Kurulum
# Bağımlılıklar
npm install

# .env.local dosyası oluştur
cp .env.example .env.local
# GOOGLE_GENERATIVE_AI_API_KEY="..." ekle

# Geliştirme sunucusu
npm run dev

Tarayıcıda http://localhost:3000
 adresini aç.
